# Introduction

Welcome to your new **AcidWiki**! This is a template designed to be fast, lightweight, and easy to customize.

## What is AcidWiki?

AcidWiki is a documentation system built with:
- **Tailwind CSS** for styling.
- **Marked.js** for markdown parsing.
- **Lucide** for icons.
- **Highlight.js** for syntax highlighting.

## Getting Started

To start using this template, you only need to:
1. Edit `config.js` to set your project name, social links, and repository.
2. Add your markdown files in the `wiki/` directory.
3. Update `wiki/structure.json` to organize your pages.
